# youkakunin — 要確認・削除候補フォルダ管理

FTP整理用：要確認・削除候補フォルダの一時保管場所

---

## 概要

ロリポップFTPサーバー上のフォルダを棚卸しするための管理ドキュメントです。  
各フォルダの用途・確認状況・対応方針を記録します。

---

## 確認ステータス凡例

| マーク | 意味 |
|--------|------|
| ⬜ 未確認 | まだ中身を確認していない |
| 🔍 確認中 | 現在調査中 |
| ✅ 確認済 | 内容を把握済み |
| 🗑️ 削除候補 | 不要と判断、削除前の最終確認待ち |
| 🔒 保持 | 削除しない（使用中または重要） |
| ✔️ 削除完了 | 削除済み |

## 削除手順（チェックリスト）

削除を実行する前に以下を確認してください。

- [ ] フォルダ内のファイルを一覧取得済み
- [ ] 稼働中サイトからの参照がないことを確認
- [ ] データベースや設定ファイルに参照がないことを確認
- [ ] バックアップが存在する（または不要と確定している）
- [ ] 関係者に確認済み
